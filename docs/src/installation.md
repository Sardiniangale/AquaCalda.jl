# Installation

AcquaCalda requires Julia 1.10 or later.

## Installing Julia

Download Julia from [julialang.org/downloads](https://julialang.org/downloads/)
or use [juliaup](https://github.com/JuliaLang/juliaup):

```bash
curl -fsSL https://install.julialang.org | sh
```

## Installing AcquaCalda

```julia
import Pkg
Pkg.add("AcquaCalda")
```

## Development version

```julia
import Pkg
Pkg.add(url="https://github.com/Sardiniangale/AquaCalda.jl")
```
