Installation

Acqua Calda requires Julia 1.10 or later.

Installing Julia

Download Julia from julialang.org/downloads or use juliaup:

curl -fsSL https://install.julialang.org | sh

Installing Acqua Calda

import Pkg
Pkg.add("AcquaCalda")

Development version

import Pkg
Pkg.add(url="https://github.com/Sardiniangale/AquaCalda.jl")
